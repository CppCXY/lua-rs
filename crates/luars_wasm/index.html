üöÄ Getting Started

These examples demonstrate how to use luars_wasm as a library in your web applications.

üì¶ Basic Usage

1. Import the WASM module:

import init, { LuaWasm } from './pkg/luars_wasm.js';

2. Initialize and use:

await init();
const lua = new LuaWasm();

// Execute Lua code
const result = lua.execute('return 2 + 2');
console.log(result); // "4"

// Evaluate expressions
const sum = lua.eval('10 * 20');
console.log(sum); // "200"

// Set global variables
lua.setGlobal('myVar', 42);

// Get global variables
const value = lua.getGlobal('myVar');
